# Autonomous Logistics Research Agent 

## Overview

AI system that automatically researches a query, gathers information from the web, and generates a structured report with supporting evidence.

---

## Features

* Automated search, scraping, and analysis
* Report generation using LLMs
* Evaluation metrics (quality, relevance, coverage)
---
## Tech Stack

Python • Streamlit • LLMs • Vector DB (Chroma/FAISS)

---

## Usage

```bash
git clone https://github.com/raidhruv/Autonomous-Logistics-Researcher-Agent
cd Autonomous-Logistics-Researcher-Agent
pip install -r requirements.txt
streamlit run app.py
```

---

## How it Works

Query → Search → Extract → Analyze → Generate Report → Evaluate
